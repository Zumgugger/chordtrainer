Chord Trainer
=============

A web-based flashcard application for learning and practicing music theory chord construction. Train yourself to instantly recognize which notes make up any chord, practice chord inversions, and hear them played back. Perfect for music students, teachers, and anyone wanting to master chord theory.

**[Deutsche Version / German Version](#deutsche-version)**

## Features

### Learning Modes
- **Chord → Notes**: Given a chord name (e.g., "C Major"), identify the notes
- **Notes → Chord**: Given a set of notes, identify the chord name
- **Mixed Mode**: Random combination of both question types
- **Competition Mode**: Timed challenges with highscore tracking

### Chord Types
- **Triads**: Major, Minor
- **Seventh Chords**: Major 7, Minor 7, Dominant 7, Minor 7♭5 (Half-Diminished), Diminished 7
- **All Circle of Fifths Keys**: C, G, D, A, E, B, F♯/G♭, D♭, A♭, E♭, B♭, F
- **Proper Enharmonic Spelling**: Automatically uses correct sharps/flats based on key (e.g., E♯ in F♯ major, not F)

### Chord Inversions
- **Root Position**: 1-3-5(-7)
- **1st Inversion**: 3-5(-7)-1
- **2nd Inversion**: 5(-7)-1-3
- **3rd Inversion**: 7-1-3-5 (seventh chords only)
- Visual display on two-octave piano keyboard

### Interactive Piano
- **Visual Feedback**: Highlighted keys show chord notes with correct enharmonic spelling
- **Audio Playback**: Click "Play Chord" or individual piano keys to hear notes
- **Two-Octave Range**: Full visual representation from low octave through middle C to high octave
- **Web Audio**: Realistic sine-wave synthesis

### Circle of Fifths
- **Interactive Visualization**: Click any key on the circle to hear it
- **Visual Highlighting**: Shows chord root and quality when answer is revealed
- **Major and Minor Keys**: Displays both parallel keys

### Customizable Practice
- **Selective Chord Types**: Filter which chord qualities to practice
- **Progress Tracking**: Automatic tracking of correct/incorrect answers
- **Local Storage**: Your progress persists across sessions
- **Export/Reset**: Export your statistics or start fresh

## How to Use

### Quick Start
1. Download or clone this repository
2. Open `index.html` in any modern web browser (Chrome, Firefox, Safari, Edge)
3. No installation or build process required - it's a single self-contained HTML file

### Practice Examples

**Example 1: Learning Major Triads**
1. Select "Chord → Notes" mode
2. Check only "Triads" in the filter
3. Click the flashcard to reveal the answer
4. View the highlighted piano keys
5. Click "Play Chord" to hear it
6. Try different inversions using the inversion buttons
7. Mark yourself correct or incorrect

**Example 2: Ear Training with Inversions**
1. Select "Chord → Notes" mode
2. Enable the chord types you want to practice
3. Click "Play Chord" before revealing the answer
4. Try to identify the notes by ear
5. Check different inversions to hear how the voicing changes
6. Flip the card to verify

**Example 3: Timed Challenge**
1. Select "Competition" mode
2. Choose your chord types
3. Answer as many as you can before the timer runs out
4. Try to beat your highscore

**Example 4: Circle of Fifths Exploration**
1. Click any key on the Circle of Fifths
2. Hear the root note
3. Practice all chords in that key
4. Notice how enharmonic spellings change (F♯ vs G♭)

### Keyboard and Mouse Controls
- **Click Flashcard**: Flip to see answer
- **Click Piano Keys**: Play individual notes
- **Click Circle Segments**: Play root notes
- **Inversion Buttons**: Change chord voicing
- **Filter Checkboxes**: Customize practice selection

## Deutsche Version

Eine webbasierte Lernkarten-Anwendung zum Erlernen und Üben von Akkordaufbau in der Musiktheorie. Trainiere dich selbst, um sofort zu erkennen, welche Noten einen Akkord bilden, übe Akkordumkehrungen und höre sie dir an. Perfekt für Musikstudenten, Lehrer und alle, die Akkordtheorie meistern möchten.

## Funktionen

### Lernmodi
- **Akkord → Noten**: Bei einem Akkordnamen (z.B. "C-Dur") die Noten identifizieren
- **Noten → Akkord**: Bei einer Notengruppe den Akkordnamen identifizieren
- **Gemischter Modus**: Zufällige Kombination beider Fragetypen
- **Wettkampfmodus**: Zeitgesteuerte Herausforderungen mit Highscore-Verfolgung

### Akkordtypen
- **Dreiklänge**: Dur, Moll
- **Septakkorde**: Major 7, Minor 7, Dominant 7, Minor 7♭5 (Halbvermindert), Vermindert 7
- **Alle Quintenzirkel-Tonarten**: C, G, D, A, E, H, F♯/G♭, D♭, A♭, E♭, B, F
- **Korrekte Enharmonische Schreibweise**: Verwendet automatisch korrekte Kreuze/♭s je nach Tonart (z.B. E♯ in F♯-Dur, nicht F)

### Akkordumkehrungen
- **Grundstellung**: 1-3-5(-7)
- **1. Umkehrung**: 3-5(-7)-1
- **2. Umkehrung**: 5(-7)-1-3
- **3. Umkehrung**: 7-1-3-5 (nur Septakkorde)
- Visuelle Darstellung auf zwei Oktaven Klaviertastatur

### Interaktives Klavier
- **Visuelle Rückmeldung**: Hervorgehobene Tasten zeigen Akkordtöne mit korrekter enharmonischer Schreibweise
- **Audio-Wiedergabe**: Klicke "Akkord abspielen" oder einzelne Klaviertasten, um Töne zu hören
- **Zwei-Oktaven-Bereich**: Vollständige visuelle Darstellung von tiefer Oktave über mittleres C bis hohe Oktave
- **Web-Audio**: Realistische Sinuswellen-Synthese

### Quintenzirkel
- **Interaktive Visualisierung**: Klicke auf beliebige Tonart im Kreis, um sie zu hören
- **Visuelle Hervorhebung**: Zeigt Akkordgrundton und -qualität bei Aufdeckung der Antwort
- **Dur- und Moll-Tonarten**: Zeigt beide parallele Tonarten

### Anpassbare Übung
- **Selektive Akkordtypen**: Filtere, welche Akkordqualitäten du üben möchtest
- **Fortschrittsverfolgung**: Automatische Aufzeichnung korrekter/falscher Antworten
- **Lokaler Speicher**: Dein Fortschritt bleibt über Sitzungen hinweg erhalten
- **Export/Zurücksetzen**: Exportiere deine Statistiken oder beginne neu

## Verwendung

### Schnellstart
1. Lade dieses Repository herunter oder klone es
2. Öffne `index.html` in einem modernen Webbrowser (Chrome, Firefox, Safari, Edge)
3. Keine Installation oder Build-Prozess erforderlich - es ist eine einzige eigenständige HTML-Datei

### Übungsbeispiele

**Beispiel 1: Dur-Dreiklänge lernen**
1. Wähle den Modus "Akkord → Noten"
2. Aktiviere nur "Triads" (Dreiklänge) im Filter
3. Klicke auf die Lernkarte, um die Antwort zu zeigen
4. Betrachte die hervorgehobenen Klaviertasten
5. Klicke "Akkord abspielen", um ihn zu hören
6. Probiere verschiedene Umkehrungen mit den Umkehrungsknöpfen
7. Markiere dich selbst als korrekt oder falsch

**Beispiel 2: Gehörbildung mit Umkehrungen**
1. Wähle den Modus "Akkord → Noten"
2. Aktiviere die Akkordtypen, die du üben möchtest
3. Klicke "Akkord abspielen" vor dem Aufdecken der Antwort
4. Versuche, die Noten nach Gehör zu identifizieren
5. Probiere verschiedene Umkehrungen, um zu hören, wie sich die Lage ändert
6. Decke die Karte auf, um zu überprüfen

**Beispiel 3: Zeitgesteuerte Herausforderung**
1. Wähle den "Competition"-Modus (Wettkampf)
2. Wähle deine Akkordtypen
3. Beantworte so viele wie möglich, bevor die Zeit abläuft
4. Versuche, deinen Highscore zu schlagen

**Beispiel 4: Quintenzirkel erkunden**
1. Klicke auf eine beliebige Tonart im Quintenzirkel
2. Höre den Grundton
3. Übe alle Akkorde in dieser Tonart
4. Beachte, wie sich enharmonische Schreibweisen ändern (F♯ vs. G♭)

### Tastatur- und Maussteuerung
- **Klick auf Lernkarte**: Umdrehen, um Antwort zu sehen
- **Klick auf Klaviertasten**: Einzelne Noten abspielen
- **Klick auf Kreissegmente**: Grundtöne abspielen
- **Umkehrungsknöpfe**: Akkordlage ändern
- **Filter-Checkboxen**: Übungsauswahl anpassen
